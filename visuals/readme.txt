# Visuals Overview

This folder contains all visuals used in the Meesho Operations Case Study.

- **avg_delivery_by_region.png** → Shows average delivery SLA (days) across regions.
- **return_rate_by_region.png** → Highlights return rate % variation by region.
- **monthly_orders_trend.png** → Order growth trend over 12 simulated months.
- **seller_onboarding_funnel.png** → Funnel visualization of seller drop-offs.
- **swot_meesho.png** → SWOT analysis diagram of Meesho.
- **dashboard_overview.png** → Combined dashboard with key KPIs for a one-glance summary.

These visuals are generated from `data/synthetic_logistics.csv` and designed in a Tableau-style format.
